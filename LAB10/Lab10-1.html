  2. If else хоёр сонголтын илэрхийлэл нь нөхцөл үнэн бол нэг үйлдлийг, худал бол нөгөө үйлдлийг тус тус гүйцэтгэнэ.
  3. Бүлэг илэрхийллүүдийг заасан тооны удаа давтаж гүйцэтгэх давталтыг тоолуурт давталт гэнэ.
  4. Давталтыг хэдэн удаа гүйцэтгэхийг урьдчилан мэдэгдээгүй үед sentinel value утгыг давталтыг зогсооход ашиглаж болно.
